# 🐼 Pandas Study Guide

A comprehensive, interactive study guide for the **pandas** data analysis library — covering 32 topics with real-world use cases, code examples, and a Jupyter notebook.

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | Interactive web guide — dark themed, searchable, syntax highlighted |
| `pandas_study_guide.ipynb` | Jupyter notebook with all examples (192 cells) |

## 🚀 How to Use

**Web Guide** — open `index.html` in any browser, no server needed.

**Notebook** — run in Jupyter:
```bash
pip install pandas numpy matplotlib jupyter
jupyter notebook pandas_study_guide.ipynb
```
